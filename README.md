# T Varaprasad Portfolio (Next.js 14)

Single-page portfolio with a sophisticated monochrome aesthetic, custom dual-layer cursor, magnetic interactions, and motion-driven sections.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Vercel Deployment (Free Tier)

1. Push this project to a GitHub repository.
2. Go to [Vercel](https://vercel.com/) and sign in with GitHub.
3. Click **Add New Project** and import the repository.
4. Keep defaults:
   - Framework preset: Next.js
   - Build command: `next build`
   - Output directory: `.next`
5. Click **Deploy**.
6. After deployment, Vercel gives a free `*.vercel.app` URL and continuous deployments on every push.

## Netlify Compatibility

This project is also deployable to Netlify using Next.js runtime support.
